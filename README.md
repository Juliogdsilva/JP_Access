## JP ACCESS

### QRCODE Example

_JP_**LOTE**_AC_**NUM**

###### JP -> Sigla restaurante

###### LOTE -> Lote do Qrcode

###### AC -> Separador (Ac de Access)

###### NUM -> Numero do QRcode

### Status QRCODE

###### waiting -> Aguardando

###### ative -> Ativo

###### used -> Utilizado

###### blocked -> Bloqueado

###### cancelled -> Cancelado
